Posicionamentodos elementos

STATIC

é o posicionamento padrão do elemento no documento HTML

RELATIVE

inicialmente não muda a posição do elemento.

é necessário usar propriedades de deslocamento (Top, Right, Bottom, Left) para movimentar o elemento.

ABSOLUTE

inicialmente o elemento sai do fluxo normal do documento HTML.

também usa as propriedades de deslocamento (Top, Right, Bottom, Left) para movimentar o elemento.

sempre é posicionado baseado na caixa que está contido e que tenha um Position diferente de Static

a caixa que contem o elemento com position absolute precisa ter um position relative - absolute - fixed para poder alinhar o elemento

Observe que o logo do instagram (com position absolute) está a flutuando no documento



FIXED

inicialmente o elemento sai do fluxo normal do documento HTML.

também usa as propriedades de deslocamento (Top, Right, Bottom, Left) para movimentar o elemento.

o elemento fica fixo em relação a tela do navegador, ou seja, não desaparece com a rolagem da tela

Observe que o logo do whatsapp (com position fixed) está fixado no lado inferior direito do documento
