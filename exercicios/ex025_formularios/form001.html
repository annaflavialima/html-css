Métodos

GET

É o método padrão para envio de formulários. Os dados preenchidos no formulário ficam visíveis na URL.

Ao submeter o formulário acima, que usa o método GET, a URL ficará da seguinte forma:

  • Esse método é usado quando trata de dados não senssíveis, como nome, peso, sexo, pesquisas...
  • A quantidade de letras no formulário não pode passar de 3 mil letras. Pois esse método suporta o envio de até 3 mil bytes.
  • Também não suporta o envio de arquivos, como imagens.

POST

É o método para envio de formulários que não informa os dados preenchidos no formulário na URL.

Ao submeter o formulário acima, que usa o método POST, a URL ficará da seguinte forma:

  • Esse método é usado quando trata de dados senssíveis, como CPF, endereço, dados do cartão, senhas...
  • Permite o envio de formulários com mais de 3 mil bytes
  • Suporta o envio de arquivos.
