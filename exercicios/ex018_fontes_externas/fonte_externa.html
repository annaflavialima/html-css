Usar fontes online

Para usar fontes diretamente do Google:

  1. Acessa o Gogle fontes
  2. Escolhe a fonte que deseja importar para o projeto
  3. Seleciona "+ select this style"
  4. Abre uma janela lateral a direita "Selected family"
  5. Seleciona a opção "@import"
  6. Copia o código descrito abaixo
  7. Cola o código no documento na parte do "head"
  8. Define a família da fonte no CSS usando a propriedade "font-family"

OBS.: ideal é usar até 03 tipos de fonte no projeto.

  1. Para grandes títulos
  2. Para destaqes
  3. Fonte normal para Corpo do texto
